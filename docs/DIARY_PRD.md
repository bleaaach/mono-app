# 日记功能 PRD

## 1. 功能概述

日记模块是 MONO 应用的核心功能之一，帮助用户记录日常生活、思考和感悟，通过时间线和日历视图展示日记记录轨迹。

## 2. 目标用户

- 希望记录日常生活的用户
- 需要情绪管理和反思的用户
- 喜欢回顾过去的用户
- 追求自我成长的用户

```typescript
interface DiaryEntry {
  id: string;
  date: string;           // YYYY-MM-DD
  content: string;
  mood?: 'sunny' | 'cloudy' | 'rainy' | 'storm' | 'moon' | 'star';
  tags?: string[];

    longitude: number;

  };

    temp: number;
  };

  bookmark?: {            // ⏳ 待实现 - 书本翻阅书签
    pageNumber: number;
    note?: string;
  };
  createdAt: string;
  updatedAt?: string;
}

// 分享记录
interface ShareRecord {
  id: string;
  diaryId: string;
  template: 'minimal' | 'journal' | 'film' | 'nature' | 'geometric' | 'mood';
  content: string;        // 分享的内容（全文/摘要）
  caption?: string;       // 配文
  imageUrl?: string;      // 生成的分享图片
  sharedAt: string;
  platform?: string;      // 分享到的平台
}

// 随机漫游配置
interface RandomWalkConfig {
  timeRange?: {           // 时间范围筛选
    start?: string;
    end?: string;
  };
  tags?: string[];        // 标签筛选
  moods?: string[];       // 心情筛选
  excludeRecent?: number; // 排除最近N天的日记
}

// 书本翻阅配置
interface BookViewConfig {
  sortBy: 'date' | 'mood' | 'tag';  // 排序方式
  groupBy?: 'month' | 'tag';        // 章节分组方式
  showBookmark: boolean;            // 是否显示书签
  pageEffect: 'curl' | 'slide' | 'fade';  // 翻页效果
}
```

## 9. 技术实现

### 9.1 存储方案 ✅

- 使用 AsyncStorage 本地存储 ✅
- 数据备份到云端（可选）⏳

### 9.2 图片存储 ✅

- ✅ 使用 expo-image-picker 选择图片
- ✅ 图片质量压缩处理 (quality: 0.7)
- ⏳ 缓存策略优化

### 9.3 搜索优化 ⏳

- 建立索引
- 使用 Fuse.js 模糊搜索

### 9.4 书本翻阅动画 ⏳

- **技术选型**
  - React Native: 使用 `react-native-page-flip` 或自定义 PanResponder + Animated
  - Expo: 使用 `react-native-reanimated` 实现高性能翻页动画
  - Web: 使用 CSS 3D transform 实现翻页效果
- **性能优化**
  - 虚拟列表渲染，只渲染当前页和前后各一页
  - 图片懒加载
  - 翻页动画使用原生驱动
- **实现要点**
  - 页面状态管理（当前页、总页数、翻页方向）
  - 手势识别与动画联动
  - 阴影和透视效果计算

### 9.5 分享图片生成 ⏳

- **技术方案**
  - 使用 `react-native-view-shot` 截取视图生成图片
  - 或使用 `react-native-canvas` 绘制分享卡片
  - Web 端使用 html2canvas
- **模板系统**
  - 定义模板配置 JSON
  - 支持动态替换文字、颜色、背景
  - 字体文件预加载

### 9.6 随机漫游算法 ⏳

- **随机策略**
  - 加权随机：近期日记权重较低，旧日记权重较高
  - 时间分散：避免连续随机到相邻日期的日记
  - 惊喜模式：优先推荐"历史上的今天"
- **缓存机制**
  - 预加载下一篇随机日记
  - 记录已漫游过的日记避免重复
