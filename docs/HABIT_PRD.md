# 习惯追踪功能 PRD

## 1. 功能概述

习惯追踪模块是 MONO 应用的核心功能之一，帮助用户建立和维持良好习惯，通过数据可视化展示习惯养成进度。

## 2. 目标用户

- 希望建立新习惯的用户
- 需要追踪日常行为的用户
- 喜欢数据可视化的用户
- 追求自我提升的用户

## 3. 功能模块

### 3.1 习惯管理

#### 3.1.1 创建习惯 ✅ 已实现
- **基本信息**
  - 习惯名称（必填）✅
  - 习惯图标（15种预设图标）✅
  - 习惯颜色（15种预设颜色）✅
  - 习惯描述（可选）✅

- **打卡设置** ✅

## 6. 数据模型

```typescript
interface Habit {
  id: string;
  name: string;
  icon: string;
  color: string;
  description?: string;
  frequency: {
    type: 'daily' | 'weekly' | 'monthly' | 'custom';
    days?: number[]; // 每周的哪几天 [1,3,5] 表示周一三五
    interval?: number; // 每隔几天
  };
  target: {
    type: 'streak' | 'count' | 'none';
    value?: number;
    deadline?: string;
  };
  reminders: {
    time: string;
    enabled: boolean;
  }[];
  category: string;
  tags: string[];
  createdAt: string;
  archived: boolean;
  paused: boolean;
  order: number;
}

interface HabitLog {
  id: string;
  habitId: string;
  date: string;
  completed: boolean;
  count: number; // 多次打卡的计数
  note?: string;
  isMakeup: boolean; // 是否为补卡
  createdAt: string;
}

interface HabitStats {
  habitId: string;
  totalCompletions: number;
  currentStreak: number;
  longestStreak: number;
  completionRate: number;
  weeklyStats: number[];
  monthlyStats: number[];
}
```
