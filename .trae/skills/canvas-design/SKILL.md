# Canvas Design

Create design philosophies and express them visually through canvas-based artwork. This skill enables the creation of logos, icons, brand identities, posters, and other visual design work with expert-level craftsmanship.

## When to Use This Skill

- Designing logos and brand identities
- Creating icons and visual assets
- Generating posters and marketing materials
- Developing visual design systems
- Creating artistic compositions
- Designing web/app graphics

## Output Formats

- **PNG** - For raster graphics and artwork
- **PDF** - For multi-page documents and print-ready designs
- **SVG** - For scalable vector graphics (icons, logos)
- **MD** - For design philosophy documentation

## Design Philosophy Creation Process

### Step 1: Define the Visual Philosophy

Create a design philosophy (4-6 paragraphs) that captures:

- **Space and form** - How elements occupy and interact with space
- **Color and material** - Color palette and textural qualities
- **Scale and rhythm** - Size relationships and visual pacing
- **Composition and balance** - Layout principles and equilibrium
- **Visual hierarchy** - How importance is communicated visually

### Step 2: Express Visually

Transform the philosophy into visual artwork:

- 90% visual design, 10% essential text
- Minimal text as visual accent
- Spatial communication over textual explanation
- Expert craftsmanship in every detail

## Design Philosophy Examples

### "Concrete Poetry"
Communication through monumental form and bold geometry.

Visual expression: Massive color blocks, sculptural typography (huge single words, tiny labels), Brutalist spatial divisions, Polish poster energy meets Le Corbusier. Ideas expressed through visual weight and spatial tension, not explanation. Text as rare, powerful gesture - never paragraphs, only essential words integrated into the visual architecture.

### "Chromatic Language"
Color as the primary information system.

Visual expression: Geometric precision where color zones create meaning. Typography minimal - small sans-serif labels letting chromatic fields communicate. Think Josef Albers' interaction meets data visualization. Information encoded spatially and chromatically.

### "Geometric Silence"
Pure order and restraint.

Visual expression: Grid-based precision, bold photography or stark graphics, dramatic negative space. Typography precise but minimal - small essential text, large quiet zones. Swiss formalism meets Brutalist material honesty.

## Essential Principles

1. **Visual First** - Ideas communicate through space, form, color, composition
2. **Minimal Text** - Text is sparse, essential-only, integrated as visual element
3. **Expert Craftsmanship** - Work should appear meticulously crafted, labored over with care
4. **Artistic Freedom** - Provide creative room for interpretive choices
5. **Professional Execution** - Nothing overlaps, proper margins, clear separation

## Logo Design Guidelines

### For Minimal/Black-White Logos:

1. **Simplicity** - Reduce to essential forms
2. **Scalability** - Works at 16px and 1600px
3. **Memorability** - Distinctive and recognizable
4. **Versatility** - Works on light and dark backgrounds
5. **Timelessness** - Avoid trendy effects

### Logo Types:

- **Wordmark** - Typography-based
- **Lettermark** - Initials or monograms
- **Pictorial** - Recognizable icon/image
- **Abstract** - Geometric forms
- **Mascot** - Character-based
- **Combination** - Icon + wordmark

## Technical Requirements

- Use appropriate fonts from `./canvas-fonts` directory
- Ensure all elements within canvas boundaries
- Maintain proper margins and breathing room
- Export at appropriate resolution for use case
- Check color contrast for accessibility

## Workflow

1. **Understand Requirements** - Logo purpose, audience, constraints
2. **Create Philosophy** - Define aesthetic direction (4-6 paragraphs)
3. **Design Iteration** - Express philosophy visually
4. **Refinement** - Polish to pristine, museum-quality craftsmanship
5. **Export** - Generate final files (PNG/PDF/SVG)

## Best Practices

- Start with black and white, add color later
- Test at small sizes
- Consider negative space
- Ensure reproducibility across media
- Document design rationale
